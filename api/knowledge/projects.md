# Projects

## CHULOOPA - Transformer-Based Drum Generation
**Technologies:** PyTorch, Audio ML, Real-time Inference, ChucK, Python, Transformers

Low-latency transformer model for real-time rhythmic generation in live performance. This is my MFA thesis project at CalArts. Researching efficient inference pipelines for generative audio systems.

GitHub: https://github.com/paolosand/CHULOOPA

## Multi-Modal Video Analysis System
**Technologies:** Gemini API, Concurrent Processing, Python, Production, Multi-Modal AI

Real-time video analysis pipeline with optimized concurrent processing for low-latency response. Built for production deployment serving thousands of users at Nuts and Bolts AI.

## Geospatial ML Pipeline
**Technologies:** PyTorch, Computer Vision, Model Optimization, CNN

Custom CNN architecture for geospatial analysis with data preprocessing and evaluation framework. Achieved >90% accuracy through model optimization. Developed during my time at Stratpoint Technologies.

## HAI - Head as Interface
**Technologies:** Arduino, Max/MSP, Wearable Computing, HCI, Real-time Audio

Wearable gestural interface for musical performance using IMU sensors. Real-time Max/MSP audio processing controlled by head movements via Arduino.

## ASCII Drone Synth
**Technologies:** MediaPipe, Tone.js, Computer Vision, Web Audio, Creative Coding

Gesture-controlled drone synthesizer for live performance using computer vision. MediaPipe hand tracking drives web audio synthesis with Tone.js.

Demo: https://paolosand.github.io/ascii_drone/

## Parallel Paths - Responsible AI in Music
